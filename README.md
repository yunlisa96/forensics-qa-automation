# 포렌식 데이터 검증 자동화

## 프로젝트 개요

포렌식 분석 과정에서 중요한 파일 무결성 검증과 메타데이터 확인 과정을 자동화해보기 위해 진행한 프로젝트이다.

실제 포렌식 도구는 분석 대상 파일이 변조되지 않았는지 확인하고, 파일 정보를 정확하게 추출해야 한다. 이러한 과정을 Python과 pytest를 이용해 자동화 테스트 형태로 구현하였다.

---

## 사용 기술

* Python 3.9
* pytest
* pytest-html
* The Sleuth Kit (TSK)
* GitHub Actions

---

## 테스트 항목

### 1. 파일 무결성 검증

SHA-256 해시값을 이용하여 파일의 무결성을 확인하였다.

* 동일 파일 반복 검증 시 동일한 해시값 반환 여부 확인
* 파일 내용 변경 시 해시값 변경 여부 확인
* 정상 파일과 손상 파일의 해시값 차이 확인

### 2. 메타데이터 검증

파일에서 추출한 메타데이터의 정확성을 검증하였다.

* 파일명 및 확장자 검증
* 파일 크기 검증
* RAW 이미지 파일 메타데이터 검증
* 실제 파일 정보와 추출 결과 비교

### 3. 예외 상황 검증

실제 분석 과정에서 발생할 수 있는 예외 상황을 테스트하였다.

* 빈 파일 처리
* 존재하지 않는 파일 접근
* 손상 파일 처리
* 대용량 파일 처리

---

## 자동화 환경

pytest를 이용해 테스트를 자동 수행하도록 구성하였으며, 테스트 결과는 HTML 리포트로 생성하여 확인할 수 있도록 하였다.

또한 GitHub Actions를 연동하여 코드 변경 시 자동으로 테스트가 수행되도록 구성하였다.
